# 圣诞主题情侣网页 🎄💑

一个充满爱意的圣诞主题手机端网页，展示情侣间的甜蜜时光和美好回忆。

## ✨ 特性

- 🎅 **圣诞主题设计** - 渐变背景、雪花飘落动画
- 📱 **移动端优化** - 完美适配手机屏幕
- 🖼️ **照片轮播** - 5 张主轮播图，支持自动播放和手势滑动
- 🎨 **照片墙** - 6 张照片网格展示，带悬停放大效果
- 💖 **丰富动画** - 包括：
  - 雪花飘落
  - 爱心跳动
  - 漂浮爱心
  - 圣诞装饰元素浮动
  - 照片缩放和旋转
  - 淡入淡出效果
- 💝 **爱的宣言卡片** - 温馨的文字表达
- 🎁 **交互功能** - 点击照片放大查看

## 🚀 使用方法

1. 直接在浏览器中打开 `index.html` 文件
2. 或使用 Live Server 等工具在本地服务器运行

## 📝 自定义说明

### 替换照片

当前使用的是占位图片，你可以：

1. 准备你们的照片（建议尺寸）：

   - 轮播图：400x500 像素（竖版）
   - 照片墙：300x300 像素（方形）

2. 将照片放入项目文件夹中（如创建 `images` 文件夹）

3. 在 `index.html` 中替换图片路径：

   ```html
   <!-- 将 -->
   <img src="https://picsum.photos/400/500?random=1" alt="回忆1" />
   <!-- 改为 -->
   <img src="images/photo1.jpg" alt="回忆1" />
   ```

### 修改文字

在 `index.html` 中找到对应部分修改：

- 标题和副标题
- 照片说明文字
- 爱的宣言内容
- 日期

### 调整颜色

在 `style.css` 中修改：

- 背景渐变色：`body` 的 `background`
- 主题色：搜索 `#764ba2` 和 `#667eea`

## 📱 支持的功能

- ✅ 触摸滑动切换照片
- ✅ 自动轮播（每 4 秒）
- ✅ 响应式布局
- ✅ 平滑滚动
- ✅ 懒加载动画
